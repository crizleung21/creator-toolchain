# Workflow Routing

## Routing Steps

1. Identify whether the user is asking for ideation, implementation, state, rules, skill work, audit, or packaging.
2. Check whether a source artifact already exists.
3. Select the narrowest workflow that can produce the requested artifact.
4. State missing inputs and phase boundaries.
5. Produce a handoff prompt.
