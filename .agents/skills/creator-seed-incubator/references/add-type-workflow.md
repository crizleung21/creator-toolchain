# Add-Type Workflow

Use `creator-seed:add-type` to define a custom project type.

```json
{
  "type_id": "custom-type",
  "display_name": "Custom Type",
  "rigor": "tight|standard|deep|creative",
  "purpose": "",
  "required_sections": [],
  "optional_sections": [],
  "default_handoff": "creator-paul-loop",
  "rule_domains": [],
  "audit_domains": [],
  "templates": []
}
```
