# Plan Apply Qualify Unify

## PLAN

Create a bounded plan with tasks, acceptance criteria, affected files, risks, and verification steps.

## APPLY

Execute tasks one at a time. Keep unrelated refactors out of scope.

## QUALIFY

For each task:

- verify observable acceptance criteria
- record evidence path
- set status
- append ledger event

## UNIFY

Close the loop:

- summarize what changed
- reconcile plan vs actual work
- propose state updates
- identify concerns
- name one next action
