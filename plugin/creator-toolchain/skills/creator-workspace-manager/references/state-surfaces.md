# State Surfaces

| Surface | Privacy | Required |
|---|---|---:|
| `workspace.json` | publishable template | yes |
| `projects.json` | local/private | yes |
| `entities.json` | private | yes |
| `state.json` | local/private | yes |
| `session-insights.json` | private | yes |
| `operator.json` | private | yes |
| `backlog.json` | local/private | yes |
| `surfaces.json` | publishable template | yes |
| `decisions.json` | local/private | yes |

Every required surface must parse as JSON and declare `schema_version`.
